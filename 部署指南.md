## 🚀 部署方案

### 方案1: GitHub Pages (推荐 - 免费)
**优点**: 免费、稳定、支持自定义域名
**步骤**:
1. 创建GitHub仓库
2. 上传 `career.chenggao.top/` 目录内容
3. 在仓库设置中启用GitHub Pages
4. 选择源分支和文件夹

**上传方法**:
```bash
git init
git add career.chenggao.top/ images/
git commit -m "上传个人网站"
git branch -M main
git remote add origin https://github.com/你的用户名/你的仓库名.git
git push -u origin main
```

### 方案2: Netlify (推荐 - 免费)
**优点**: 自动部署、支持表单、全球CDN
**步骤**:
1. 注册Netlify账号
2. 拖拽 `career.chenggao.top` 文件夹到部署区域
3. 自动生成域名: `随机名字.netlify.app`
4. 可绑定自定义域名

### 方案3: Vercel (推荐 - 免费)
**优点**: 速度快、自动HTTPS、支持无服务器函数
**步骤**:
1. 注册Vercel账号
2. 导入GitHub仓库或直接上传
3. 自动部署完成

### 方案4: 传统虚拟主机
**要求**: 支持静态网站的虚拟主机
**上传方法**:
- 将 `career.chenggao.top/` 下所有文件上传到主机根目录
- 将 `images/` 目录也上传到根目录

## 🌐 自定义域名设置

### 购买域名后
1. **DNS配置**:
   - A记录: `192.168.1.1` (服务商提供)
   - CNAME记录: `www` 指向主域名

2. **HTTPS证书**:
   - GitHub Pages/Netlify/Vercel 自动提供
   - 虚拟主机需要申请Let's Encrypt证书
